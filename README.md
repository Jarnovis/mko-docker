# mko-docker
In deze repositorie worden de basics van Docker uitgelegd en is er een self-hosted GitHub runner docker image beschikbaar gesteld.

Deze opdracht wordt gedaan vanuit de Haage Hogeschool Innovative Development (ID).

Alle documentatie (markdown) wordt via een workflow automatisch omgezet naar HTML. De output hiervan is te vinden op [deze link](https://jarnovis.github.io/mko-docker/).
